#Multi-view Graph Clustering with Granular Generation Chain

### Run the main program

To run the main script, simply execute:
main_GCDC.m --Run the discrete multi-view clustering algorithm GCDC
main_GCFC.m --Run the fuzzy multi-view clustering algorithm GCFC

### CVI
The procedure regarding the clustering evaluation index.

### data
Six multi-view datasets:
3Sources; 100leaves; MSRC

### FGCG
Fast Granularity Generation Chain (GGC) generation algorithm (FGCG) and its granularity division.

### Pre-treatment
Programs related to normalization and generating graph data.

### MSA
GGC alignment and metrics

### GCDC
The main body of the GCDC clustering algorithm.

### GCFC
The main body of the GCFC clustering algorithm.
